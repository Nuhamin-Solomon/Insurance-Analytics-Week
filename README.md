# Insurance Risk Analytics Project

This project analyzes risk, claims, and profitability for an insurance portfolio.  
It includes:

- Task 1: EDA & Statistical Understanding  
- Task 2: DVC reproducible data pipeline  
- Task 3: Hypothesis testing  
- Task 4: Predictive modeling for risk-based pricing  

All work follows Git best practices with branches, commits, and pull requests.
